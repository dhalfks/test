String : 가장 많이 사용되는 데이터 타입

  • 백슬래시(\)는 코드로 인식될 수 있는 문자를 문자로 변환시키는 역할
  • 문자열은 index 번호를 가지고 있음. (0부터 시작)
  • 문자열 데이터를 받을 때 문자열 값 내에서 개행을 추천하지 않음
  • 문자열 길이를 구할 때 : length 번호를 나타낼때는 : length - 1 (마지막 index)
  • 문자 추출 관련 메서드 : slice(), substring(), substr()
  • 문자 대체 메서드
    • replace(oldstr, newstr)
    • trim() : 문자열 양쪽 공백제거
    • padStart(길이, 문자) : 앞쪽에 길이만큼 문자 채우기
    • padEnd(길이, 문자) : 뒤쪽에 길이만큼 문자 채우기
    • 문자값의 힌트, 전체값 중 일부만 보여줄때 등.
  • 문자 추출 메서드 : charAt(index) : index 번호의 문자를 추출
  • 탐색 메서드
    • indexOf("str") : 해당 문자열의 index 번호 추출
    • lastIndexOf("str") : 해당 문자열의 index 번호 추출 (뒤에서 부터 탐색)
  • startWith(), endWith() : 지정문자열이 탐색 대상문자열의 시작/끝에 있는지 확인 : true / false
    파라미터 값 1 : (지정한문자열)의 시작과 끝에 존재하는지를 확인
    파라미터 값 2 : startWith(지정문자열, 시작 index)
